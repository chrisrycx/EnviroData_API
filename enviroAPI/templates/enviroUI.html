Environmental Data Quality API - Demonstration

This page provides a simple demonstration of an environmental data quality API. Data (in this case 2 meter air tempertures) are submitted to the API as pairs of dates and values. The values are assessed and a data quality rating between 0 and 2 is returned.

Quality Ratings:

  • 0 - Good
  • 1 - Suspicious
  • 2 - Bad

Assess 2m Air Temperature

Click "check data" to assess the air temperature data in the input. The default input data can be changed if desired but must follow the same format and be limited to X number of rows. Results are plotted below.
